# Priya | Full Stack Developer Portfolio

Live site: https://portfolio-priya28.vercel.app

A personal portfolio website showcasing my projects, certificates, and skills as a Full Stack Developer specializing in Java Spring Boot and React.

## Features

- Responsive single-page design with smooth scroll navigation
- Light/dark theme toggle
- Animated typing effect, scroll-reveal sections, and skill progress bars
- Projects section with live demo and GitHub links
- Certificates section showcasing completed courses and virtual internships
- Contact form (opens the visitor's email client with a prefilled message)

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Font Awesome icons, Google Fonts (Inter)

## Projects Featured

- **Connectify** — real-time chat app (Java, Spring Boot, WebSocket, Thymeleaf)
- **AI Interview Coach** — AI-powered interview prep tool (React, Vite)
- **Task Manager** — task management app (JavaScript, HTML, CSS)
- **Weather Dashboard** — weather app with live forecasts (React, Vite)
- **Expense Tracker** — full stack expense tracker with auth, budgets, and analytics dashboard (React, Node.js, Express, PostgreSQL)

## Running Locally

This is a static site — no build step required.

```bash
git clone https://github.com/Priyanka28-create/portfolio.git
cd portfolio
python3 -m http.server 8080
# open http://localhost:8080
```

## Deployment

Deployed on [Vercel](https://vercel.com), auto-deploys on every push to `main`.
